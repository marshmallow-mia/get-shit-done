# Example: Decompiling a Classic Platformer

This example demonstrates decompiling a hypothetical classic 2D platformer game from the 1990s.

## Scenario

**Game:** "Super Jump Adventure" (fictional example)
- **Platform:** Windows 95
- **Release Year:** 1995
- **Binary:** superjump.exe (512 KB)
- **Goal:** Generate compilable source code for preservation and porting

## Expected Output Structure

After decompilation, you would get:

```
features/decompiled_source/super-jump/
├── CMakeLists.txt
├── README.md
├── include/
│   ├── types.h
│   ├── game.h
│   └── renderer.h
└── src/
    ├── main.c
    ├── game.c
    ├── rendering/
    ├── physics/
    └── levels/
```

See full example in the standalone package documentation.
